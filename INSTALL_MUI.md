# Instalar Material UI

Execute este comando para instalar as dependências:

```bash
npm install @mui/material @emotion/react @emotion/styled @mui/icons-material @iconify/react
```

Dependências:
- @mui/material - Componentes Material UI
- @emotion/react + @emotion/styled - Sistema de estilos do MUI
- @mui/icons-material - Ícones Material
- @iconify/react - Ícones Iconify (mais variedade)

Após instalar, execute:
```bash
npm run dev:mock
```
